# 欢迎来到我的博客

你好！这是自动生成的第一篇文章，标志着你的博客已经成功搭建完成。

## 🎉 博客功能

你的新博客支持以下功能：

- ✅ **文章编辑** - 在管理后台使用 Markdown 编辑器创作
- ✅ **保存与更新** - 自动保存，随时修改
- ✅ **Markdown 导入** - 支持导入现有的 Markdown 文件
- ✅ **公网访问** - 通过隧道服务实现外网访问
- ✅ **响应式设计** - 支持手机和电脑浏览

## 📝 如何写作

1. 访问**管理后台**（点击页面右下角的"管理后台"按钮）
2. 在编辑器中输入文章标题和内容
3. 使用 Markdown 语法格式化你的文章
4. 点击"保存文章"即可发布

## 🎨 Markdown 语法示例

```markdown
# 标题
## 二级标题

这是普通段落，支持 **粗体** 和 *斜体*。

- 列表项 1
- 列表项 2
- 列表项 3

> 这是引用文字

[链接文字](https://example.com)
```
