🎉 博客功能

你的新博客支持以下功能：

  • ✅ 文章编辑 - 在管理后台使用 Markdown 编辑器创作
  • ✅ 保存与更新 - 自动保存，随时修改

📝 如何写作

  1. 访问管理后台（点击页面右下角的"管理后台"按钮）
  2. 在编辑器中输入文章标题和内容
  3. 使用 Markdown 语法格式化你的文章
  4. 点击"保存文章"即可发布
